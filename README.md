# 🏥 Doctor Appointment Management System

A web-based application built with PHP & MySQL to automate 
doctor appointment booking and management.

## 🚀 Features
- Doctor Dashboard — View, Approve & Cancel appointments
- Generate Reports by date range
- Search patients by Name / Mobile / Appointment Number
- Users can Book & Check appointment status (No login needed)

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, Bootstrap
- **Backend:** PHP 5.6+
- **Database:** MySQL
- **Server:** Apache (XAMPP)

## ⚙️ How to Run
1. Install XAMPP
2. Extract project into `htdocs` folder
3. Import SQL file in phpMyAdmin
4. Start Apache & MySQL
5. Open `http://localhost/dams/`
